TheOwlDude Home

Discover the secret ordered colored code by refining a series of guesses. To construct your guess use the arrow keys. The left and right keys change the selected guess element, reflected by the black square. Up and down keys cycle through the available colors. When ready press the button labeled 'Guess'. The closeness of your guess to the actual secret code will be indicated by small black and gray circles. Black circles means the guess contained a correct color in the correct place. Gray circle indicate a correct color but incorrect place. The game also has a couple of 'cheat' modes. Check the box labeled 'Consistent Codes' and the list of codes consistent with the game information so far will be displayed. Check the box labeled 'Best Guess' and every possible guess will be listed ordered by the quality of the guess, best guess on top. The guesses' quality scores will also be displayed. The score represents the expected number of consistent unguessed codes remaining after that guess is made. Warning: even for the relatively small game size of code length 4 color count 6, calculating best guess is expensive early in the game and there is a noticeable delay. For large games Best Guess can take a long time.
